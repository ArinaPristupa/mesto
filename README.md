#Место

Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки. Сделано по макету из Фигмы.

**Figma**

* [Ссылка на макет в Figma](https://www.figma.com/file/2cn9N9jSkmxD84oJik7xL7/JavaScript.-Sprint-4?node-id=0%3A1)

Есть index.html , index.css ,  index.js, директория blocks , файлы с картинками в папке images и README.md. В проекте присутствуют медиазапросы, реализована адаптивная верстка. Создана файловая структура по БЭМ (Nested). Все ссылки и интерактивные элементы имеют состояние наведения :hover. Контентные изображения имеют alt с корректным описанием, которое соответствует языку страницы. Используется язык программирования JavaScript. Технология DOM.
